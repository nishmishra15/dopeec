# DOPEEC
Codes related to formulating a DRL agent for optimizing indoor PM exposure and energy consumption

# Description of Excel sheet data
1. The Excel sheet attached here contains indoor PM emission rates, deposition rates, and ambient temperature, RH, and PM concentrations for 20 days.
2. The data present in this file, and other datasets derived from it have been used to train multiple DQN models.
3. Before using this data for any publication, the consent from the auhtors are required. You may write to sameer.patel@iitgn.ac.in to get the necessary permission.
